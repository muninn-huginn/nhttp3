nhttp3

Production-grade QUIC + HTTP/3 implementation in Rust. Built from scratch, with Python and WASM bindings.

198 tests passing
7 RFCs implemented
8 crates
Zero unsafe
cargo add nhttp3

8 crates, one stack

nhttp3-quic

QUIC Transport

Packets, frames, TLS 1.3 via rustls, streams with AsyncRead/Write, flow control, loss detection, NewReno + CUBIC + BBR congestion control, QLOG.

nhttp3-h3

HTTP/3 Protocol

Frame codec (DATA, HEADERS, SETTINGS, GOAWAY), pseudo-header validation, HTTP crate type integration, error codes per RFC 9114.

nhttp3-qpack

QPACK Compression

Static table (99 entries), dynamic table with eviction, encoder/decoder with 50% compression on realistic headers.

nhttp3-python

Python Bindings

PyO3 with custom async bridge (no pyo3-asyncio). Background tokio runtime, call_soon_threadsafe completions.

nhttp3-wasm

WASM Target

wasm-bindgen for browser (WebTransport adapter) and non-browser runtimes (Cloudflare Workers, Deno).

nhttp3-ffi

C FFI Layer

Opaque handle types, callback-based async, error codes. Tokio runtime managed internally.
