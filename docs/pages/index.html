// modules

8 crates, one stack

nhttp3-quic

QUIC Transport

Packets, frames, TLS 1.3 via rustls, streams, flow control, loss detection, NewReno + CUBIC + BBR, QLOG, endpoint I/O loop.

nhttp3-h3

HTTP/3 Protocol

Frame codec, pseudo-header validation, HTTP crate types, error codes per RFC 9114.

nhttp3-qpack

QPACK Compression

Static table (99 entries), dynamic table, encoder/decoder. 50% compression on realistic headers.

nhttp3-python

Python Bindings

PyO3 + custom async bridge. No pyo3-asyncio. Tokio runtime on background thread.

nhttp3-wasm

WASM Target

wasm-bindgen. Browser via WebTransport, non-browser via raw UDP.

nhttp3-ffi

C FFI Layer

Opaque handles, callback-based async, error codes. Runtime managed internally.

// integrations

Drop-in for any framework

FastAPI

1 line change

nhttp3.serve(app, port=4433)
Replaces uvicorn.run(). Full ASGI support. Streaming SSE, WebSockets coming.

SGLang

LLM Serving

OpenAI-compatible /v1/chat/completions with token streaming. No HOL blocking = faster token delivery to clients.

Ollama

HTTP/3 Proxy

Front Ollama's REST API with HTTP/3. Streaming /api/generate and /api/chat. Edge inference friendly.

Express / Node.js

QPACK + Frames

WASM bindings for header compression (50% savings) and HTTP/3 frame encoding. Router with frame output.

// clients

Client access

curl

curl --http3

curl --http3 https://localhost:4433/ -k

Parallel multiplexing:
curl --http3 --parallel url1 url2 url3 -k

Requires curl 7.88+ with HTTP/3 support.

Browser

Fetch API + WebTransport

Browsers auto-negotiate HTTP/3 via Alt-Svc header. Standard fetch() works — Chrome/Firefox/Safari all support h3.

WebTransport API for bidirectional streams and datagrams (Chrome 97+).

Python

nhttp3 client

conn = await ep.connect("host", 443)
send, recv = await conn.open_bidi_stream()

Async/await native. Multiplexed streams without HOL blocking.
