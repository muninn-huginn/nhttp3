Express / Node.js

Use nhttp3's WASM bindings for QPACK header compression and HTTP/3 frame encoding in Node.js.

Install

cd nhttp3-wasm && wasm-pack build --target nodejs
npm link ./pkg

QPACK Compression

const nhttp3 = require('nhttp3');

const headers = [
  [':method', 'GET'],
  [':path', '/api/users'],
  ['accept', 'application/json'],
];

const encoded = nhttp3.encode_headers(headers);
const decoded = nhttp3.decode_headers(encoded);
console.log(`Compressed: ${encoded.length} bytes (was ${rawSize})`);

Frame Encoding

const body = new TextEncoder().encode('{"ok":true}');
const dataFrame = nhttp3.encode_data_frame(body);
const settingsFrame = nhttp3.encode_settings_frame();
