Ollama HTTP/3 Proxy

Front Ollama's REST API with HTTP/3 for mobile/edge clients. Streaming /api/generate and /api/chat without HOL blocking.

Setup

# Start Ollama normally
ollama serve

# Start the HTTP/3 proxy
python proxy.py --ollama-url http://localhost:11434

Usage

# Generate (streaming)
curl --http3 https://localhost:4433/api/generate \
  -d '{"model":"llama3","prompt":"Why is the sky blue?"}' -k --no-buffer

# Chat
curl --http3 https://localhost:4433/api/chat \
  -d '{"model":"llama3","messages":[{"role":"user","content":"Hi"}]}' -k --no-buffer

# List models
curl --http3 https://localhost:4433/api/tags -k
